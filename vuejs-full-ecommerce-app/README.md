# Commerce.js Integration Example
## Project setup
```
npm install
```

### Copy example env and add chec credentials
```
cp example.env .env
```

### Compiles and hot-reloads for development
```
npm run serve
```

### Compiles and minifies for production
```
npm run build
```

### Lints and fixes files
```
npm run lint
```
